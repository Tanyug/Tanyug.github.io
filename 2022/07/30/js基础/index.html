两个重要的 JavaScript 关键字

let 和 const。
使用 var 关键字声明的变量不具备块级作用域的特性，它在 {} 外依然能被访问到。
let 声明的变量只在 let 命令所在的代码块 {} 内有效，在 {} 之外不能访问。let 声明的变量不会在作用域中被提升
const 声明一个只读的常量，一旦声明，常量的值就不能改变。
